Teiid 12.1.2 Released

23 Sep, 2019 1 minute read

< Back

Teiid 12.1.2 is now available for download and addresses several important issues.

Teiid 12.1.2 addresses 7 issues:

  • [TEIID-5734] - Planning errors with masking due to unaliased column references
  • [TEIID-5739] - Procedure does not get executed (without error) when comment and variable declaration are present
  • [TEIID-5765] - Criteria Pushdown is not working on FULL OUTER JOIN
  • [TEIID-5783] - OUTER JOIN yields wrong results when used with GROUP BY, CASE and COUNT DISTINCT
  • [TEIID-5804] - is distinct from results in TEIID30167 could not find symbol
  • [TEIID-5805] - information_schema.views returns only tables instead of views
  • [TEIID-5807] - Multiple levels vdb-importing results in error

This is the last community release on the 12.1.x line - please move to 12.2.1 or 12.3.0.

Thank you, Steve
